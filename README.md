  ### Install NPM packages:

    npm install

   ### Start the project:

    npm start

## 📌 About the project
This project is a clone of the popular 2048 puzzle game.
The main goal was to practice JavaScript logic, DOM manipulation, and event-driven programming.

---

## ⚙️ Game mechanics

- Tiles move using arrow keys (↑ ↓ ← →)
- Equal tiles merge into one (2 → 4 → 8 → ...)
- A tile cannot merge twice in one move
- Each move generates a new tile (2 or 4)
- Game ends when no moves are available

---

## ✨ Features
- 4x4 game grid
- Keyboard controls (arrow keys)
- Tile merging logic
- Random tile generation (2 / 4)
- Score tracking
- Win / Game Over states
- Restart functionality
